Potential Calculation Programm

There is a programme available in this group to obtain the electrostatic potential from trajectory data.

It uses the following input:

  1. PSF file name
  2. Number of DCD files (with the following in sequence):
# Name of DCD file
# Number of DCD frames (per file or patch)
  1. Number of residues of interest
  2. Names of these residues
  3. Width of the z bins
  4. Limits of the integration
  5. Residue name of the lipids
  6. Switch to split lipids into 3 head parts or keep it in one piece (yes=1 or no=0)

For further information, please have a look into the attached PDF file.
