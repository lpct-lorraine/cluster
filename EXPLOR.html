Pour changer votre mot de passe  :

Une fois connecté au noeud d'accès du cluster, vous pouvez utiliser la commande passwd

Connexion au cluster :


Pour vous connecter au noeud d'accès du cluster à partir d'un PC sur le réseau 193.50.239.1 à 193.50.239.253 : 

ssh -Y user_name@frontale1.explor.univ-lorraine.fr


Utilisation du cluster :

Le cluster est sous slurm, voir la page dédiée slurm_cluster_explor.


Les logiciels sont installés sous forme de modules gérés par l'application module de sourceforge : http://modules.sourceforge.net/ ( (cf documentation jointe).

Pour voir les modules disponibles : module av

Monitoring de l'utilisation des noeuds: http://ganglia.explor.univ-lorraine.fr/

Files d'attente :

Il y a une file d'attente defq qui pour l'instant est configurée sans restriction.


Zones de travail :

Votre zone de travail /home/login a un quota de 300 Go par défaut. Cette zone n'est pour l'instant pas sauvegardée.
