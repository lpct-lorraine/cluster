Modeller

Software for generating homology models of proteins: older than alphafold2, but sometimes useful.

Licence

Web site:
https://salilab.org/modeller/
Free academic licens, an inscription to the site is required. A license key will be delivered by mail.

Installation

Download modeller-10.2.tar.gz from the page: https://salilab.org/modeller/download_installation.html

by clicking on  the linlk "Generic Unix tarball".

Installation is explained in the chapter "For generic Unix" on the page: https://salilab.org/modeller/10.2/release.html#deb

  1. Open a console or terminal (e.g. xterm, Konsole, GNOME terminal) and change to the directory where you downloaded the .tar.gz file. Unpack the file with the following commands: gunzip modeller-10.2.tar.gz tar -xvf modeller-10.2.tar
  2. Go to the ./modeller-10.2 directory and run the installation script: cd modeller-10.2 ./Install
  3. You can define where the binary is installed when running ./Install
  4. You have to provide the license key you received.
