Running with a SLURM script

Exemple de script "submit_slurm.sh" : 

[fpascale@visu8 test_terachem]$ cat start.631.sp
# basis set: 6-31G
basis		6-31++g
# coordinates file
coordinates	caffeine.xyz
# molecule charge
charge          0
# SCF method (rhf/blyp/b3lyp/etc...)
method		b3lyp
dftgrid         5
# add dispersion correction (DFT-D)
dftd		no
# type of the job (energy/gradient/md/minimize/ts): single-point energy
#run		energy
run		minimize
gpus            2
# timings
timings         yes
end


#!/bin/bash
#
#SBATCH --comment=terachem-test
#SBATCH --ntasks=20
#SBATCH --job-name=terachem-test
#SBATCH --output=output.%j.terachem-test
#SBATCH --error=output.%j.terachem-test
#SBATCH --time=01:00:00
#SBATCH --gres=gpu:4
#SBATCH -p k80
#SBATCH -N 1

module purge
module load TeraChem/std

terachem start.631.sp

Soumission avec la commande : 

sbatch submit_slurm.sh
