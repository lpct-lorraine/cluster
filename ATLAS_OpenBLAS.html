ATLAS / OpenBLAS


Description

ATLAS (Automatically Tuned Linear Algebra Software) provides highly optimized Linear Algebra kernels for arbitrary cache-based architectures. ATLAS provides ANSI C and Fortran77 interfaces for the entire BLAS API, and a small portion of the LAPACK AP

ATLAS is outdated. The newer library OpenBLAS is far more efficient. See for example the benchmark https://github.com/tmolteno/necpp/issues/18

To compile OpenBLAS,

git clone https://github.com/xianyi/OpenBLAS.git

cd OpenBLAS

make USE_OPENMP=1

make install PREFIX=~/lib

To link to your code

-L ~/lib -lopenblas
