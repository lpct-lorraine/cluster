Nouveau code


Étapes pour la mise en place d'un code de calcul ou d'une version de code

  • Un scientifique fait une demande de mise en service d'une version d'un code à la cellule calcul scientifique.
  • Récupération des sources ou d'une version d'évaluation.
  • Mise en place sous forme de module dans le répertoire /auto/soft/test/nom_du_code/compilateur/version
  • Le chercheur teste et valide le code sur les nœuds de calcul à sa disposition en chargeant le module préparé
  • Dans le cas de code payant, passer pas la cellule de gestion
  • En cas de validation, information de la disponibilité du code pour tout le laboratoire par exemple via une page web ou un mail d'information.
