vpn under Linux machine

The vpnc program is used to connect to the "Universit√© de Lorraine" Cisco VPN.

Installation

  • Under Redhat-like
sudo yum -y install vpnc


  • Under Debian-like
sudo apt-get install vpnc


Configuration

Add a new file with "sudo vi /etc/vpnc/ul.conf" 

IPSec gateway vpn.lothaire.net
IPSec ID univ-lorraine
IPSec secret LeVpnPourL'UL
Xauth username pascale5@lpct


Remplacer pascale5 par son login de messagerie

Running a vpn session

fpascale@system:~/# sudo vpnc ul
Enter password for pascale5@fst-srsmc@vpn.lothaire.net:

# Result :
VPNC started in background (pid: 31662)...

# Try ssh to a TMS machine:
fpascale@system:~$ ssh fpascale@193.50.239.70

Stopped a vpn session

fpascale@system:~/# sudo vpnc-disconnect
