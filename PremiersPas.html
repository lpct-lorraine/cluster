Premiers pas dans le système informatique du LPCT

User access

Each user should have an account on the laboratory ldap. On suppose que l'utilisateur possède un compte informatique et se connecte depuis une machine Linux ou MacOS ou Windows.

Cluster access

Depuis une machine de bureau au LPCT, ou à l'extérieur en utilisant le VPN, on devra d'abord se connecter sur la machine (appelée 'frontale').

Voir enusite la page dédiée à SLURM slurm.


Configuration du VPN

remote graphical access with x2goclient

You need to install the x2goclient on your PC.

Quick configuration :

  • click on session menu and "New session"
  • Enter a session name like "visuX", Host : visuX.pct.site.univ-lorraine.fr, Login: "uor_login", Session type: "XFCE" and don't touch ther parameters
  • click on "connection sheet", and push the cursor from ADSL to LAN
  • Don't touch the configuration of "Input/Output" sheet
  • On "Media sheet", deselect "Enable sound support" and "client side printing support"
  • On "Shared folders" deselect "Use SSH port forwardinf to tunnel..."
  • To save your configuration click on "OK" button

You need to install x2goclient package on you system to get this kind of access.

Sometimes, you need to redefine the layout keyboard under the x2goclient session with this kind of command : 

setxkbmap -layout us -option grp:shifts_toggle -option grp_led:caps

Montage fichiers

espace BUL

sudo mount -t davfs https://bul.univ-lorraine.fr/remote.php/webdav/ dav/

filer

  • espace personnel :
sudo mount -t cifs //pers.ad.univ-lorraine.fr/pascale5 /home/fpascale/smb -o "user=pascale5,domain=ul,vers=1.0"
  • espace commun :
sudo mount -t cifs //pers.ad.univ-lorraine.fr/labo_commun /home/fpascale/smb -o "user=pascale5,domain=ul,vers=1.0"
