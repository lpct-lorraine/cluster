Lecteur PDF : okular

Génération de la vidéo : 

ffmpeg -i image011-%02d.png  -pix_fmt yuv420p movie.mp4

Exemple de code latex 

\documentclass{beamer}

\usepackage{multimedia}

\begin{document}

\begin{frame}{Embedded Animation}

\movie[height = 0.6\textwidth, width = 0.8\textwidth,autostart, poster, showcontrols=false,loop]{}{movie.mp4}

\end{frame}

\end{document}
