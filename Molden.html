Installation 

tar -xvf molden5.7.tar.gz -C /opt/
cd /opt/molden5.7
make gmolden

Modifier le fichier : bash.bashrc pour ajouter un chemin direct à l'application. 

$cd /etc
$sudo nano bash.bashrc

soit en chemin direct : 

export PATH=$PATH:/opt/molden5.7

soit à la suite de vos chemins existants : 

export PATH=$PATH:/opt/vmd/bin/:/opt/molden5.7

Pensez à modifier les droits sur le dossier Molden5.7 : 

# cd /opt/
opt# chown root.root molden5.7/
opt# chown root.root -R molden5.7/
opt# chmod oug+x molden5.7/
