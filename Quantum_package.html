Installation

https://github.com/QuantumPackage/qp2/blob/master/INSTALL.rst 

module load compiler/gnu-5.2.0/std
mkdir qp2
cd qp2
wget https://github.com/QuantumPackage/qp2/archive/refs/tags/2.1.2.tar.gz

tar -zxf 2.1.2.tar.gz -C /auto/soft/qp2
cd /auto/soft/qp2
mv qp2-2.1.2/ 2.1.2/
cd 2.1.2
./configure

Requirements

  x  Linux OS
  x  Fortran compiler : GNU Fortran, Intel Fortran or IBM XL Fortran
  x  `GNU make`_
  x  `Autoconf`_
  x  `Python`_ > 2.6
  x  |IRPF90| : Fortran code generator
  x  |EZFIO| : Easy Fortran Input/Output library generator
    |BLAS| and |LAPACK|
    Zlib
  x  `GNU Patch`_
  x  |ZeroMQ| : networking library
  x  GMP : Gnu Multiple Precision Arithmetic Library
  x  |OCaml| compiler with |OPAM| package manager
  x  Bubblewrap : Sandboxing tool required by Opam
  x  libcap : POSIX capabilities required by Bubblewrap
  x  |Ninja| : a parallel build system

for elt in irpf90 ninja libcap zeromq f77zmq gmp libcap bwrap ocaml docopt resultsFile bats; do ./configure --install $elt; done


module load openmpi/3.0.0_gnu-5.2.0
./configure -c config/gfortran_mpi.cfg

make

Installation rapide

module load compiler/gnu-5.2.0/std
git clone https://github.com/LCPQ/quantum_package.git
cd quantum_package
./configure  --production ./config/gfortran.cfg
source ./quantum_package.rc 
qp_module.py install Full_CI Hartree_Fock MRCC_CASSD All_singles CAS_SD
ninja
make -C ocaml
cd tests ; ./run_tests.sh
