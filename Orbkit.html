Built the python env

export INSTALLPATH=/auto/soft/orbkit/

cd $INSTALLPATH

virtualenv -p /usr/bin/python3.6 $INSTALLPATH/venv_python3.6

source $INSTALLPATH/venv_python3.6/bin/activate

pip install --upgrade pip

Install orbkit

https://orbkit.github.io/install.html

Prerequist : mayavi Latest stable release

As of the latest release, i.e. 4.6.0 and above, if you are using Python 3.x and are on a 64 bit machine, installation via pip is the easiest and is as follows: 

pip install mayavi

pip install PyQt5

Other prerequisites

pip install h5py scipy cython

orbkit installation

git clone https://github.com/orbkit/orbkit.git git

export ORBKITPATH=/auto/soft/orbkit/git

cd $ORBKITPATH

python setup.py build_ext --inplace clean

export PYTHONPATH=$PYHONPATH:$ORBKITPATH

export PATH=$PATH:$ORBKITPATH/tools

Pour QT5, il est nécessaire d'ajouter des polices supplémentaires : 

sudo yum update freetype-devel
