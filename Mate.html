Mate


Ajout de MATE à Ubuntu Desktop 12.04LTS

Note : gnome-session-fallback installé (= choix Unity ou Gnme Classic pour la session X)


installation du Gestionnaire de connexion 

sudo apt-get install gdm (pas fait)

ou 

sudo apt-get install lightdm

(déjà installé par défaut par Unity)

ou 

sudo add-apt-repository ppa:nilarimogard/webupd8sudo 

sudo apt-get update 

sudo apt-get install mdm

mint-mdm-themes (pas fait) 

sudo mdmsetup

(personnalisation de mdm) (Mate Display manager)



Choix du gestionnaire de connexion (si besoin)

sudo dpkg-reconfigure mdm

COMMENCER ICI

Préparation à l’installation de Mate

apt-add-repository ppa:ubuntu-mate-dev/ppa

apt-add-repository ppa:ubuntu-mate-dev/precise-mate   (trusty-mate pour 14.04LTS)

apt-get update && sudo apt-get
upgrade
